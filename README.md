# flutter-on-vps

Open your terminal.
Copy and paste these commands.

```
cd ~
wget https://github.com/thecreatorx000/flutter-on-vps/archive/master.zip
unzip master.zip
rm master.zip
mv ~/flutter-on-vps-master/* ~
rm flutter-on-vps-master -r
source ./main.sh
```
**Press Y or whatever suggested in output**
